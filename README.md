# QuizHack - Simulatore SMAT

Simulatore HTML per prepararsi alla prova scritta SMAT per i profili:

- **B - Addetto/a Specializzato/a Ricerca Perdite di Rete**
- **C - Tecnico/a Meccatronico/a e Automazione**

## Link GitHub Pages

Quando GitHub Pages è attivo, il simulatore è disponibile qui:

https://ngc993.github.io/quizhack/

## Funzionalità

- 500 domande a risposta multipla
- Sezioni per argomento:
  - Sicurezza sul lavoro - D.Lgs. 81/08
  - Automazione industriale, PLC, SCADA e OT
  - Ricerca perdite, reti idriche e idraulica di base
  - Logica, matematica e ragionamento
  - CCNL Gas-Acqua, lavoro e organizzazione
- Correzione immediata in modalità studio
- Evidenza verde per risposta corretta
- Evidenza rossa per risposta errata
- Dashboard con risposte, corrette, errate e avanzamento
- Filtro per sezione
- Mescolamento domande
- Modalità simulazione da 40 domande
- Correzione visibile solo dopo invio finale della simulazione
- Esito finale superato/non superato con soglia 28/40
- Header della simulazione sticky
- Layout responsive mobile e desktop
- File statico senza backend

## Uso consigliato

1. Studia per sezione usando la modalità normale.
2. Usa il filtro per concentrarti sulle aree più deboli.
3. Quando sei pronto, avvia **Simulazione 40 domande**.
4. Rispondi senza vedere le soluzioni.
5. Invia la simulazione e controlla il punteggio.
6. Ripeti finché raggiungi stabilmente almeno **32/40**.

## Soglia simulata

La soglia minima impostata è **28/40**, coerente con il punteggio minimo indicato nel bando per l’accesso alla fase successiva.

## Deploy GitHub Pages

Per attivare GitHub Pages:

1. Vai su `Settings` del repository.
2. Apri `Pages`.
3. Source: `Deploy from branch`.
4. Branch: `main`.
5. Folder: `/ (root)`.
6. Salva.

Il sito sarà pubblicato su:

```text
https://ngc993.github.io/quizhack/
```

Il simulatore è pensato per esercitazione personale. Le domande sono ricostruite sulla base degli argomenti più probabili del bando e non rappresentano domande ufficiali SMAT.
